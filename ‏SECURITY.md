# Security Policy

## Supported Versions
Swiss Army Knife is currently in active development. Security patches and stability improvements are applied to the `main` branch.

## Reporting a Vulnerability
We take the security and ethical use of Swiss Army Knife seriously. If you discover a security vulnerability, please follow responsible disclosure practices:

1. **Do not open a public issue.**
2. Send a detailed report to **[your-email@example.com]** *(replace with your actual contact)*.
3. Include:
   - Clear steps to reproduce
   - Potential impact assessment
   - Suggested mitigation (if applicable)
4. We will acknowledge receipt within 48 hours and provide a technical assessment within 7 days.
5. Coordinated disclosure will be managed mutually. Public credit will be provided upon your request.

## Ethical Usage Notice
This tool is designed for authorized security testing, bug bounty hunting, and defensive research. Users must comply with all applicable laws, terms of service, and responsible disclosure guidelines. Unauthorized scanning or exploitation is strictly prohibited.

## Security Best Practices
- Always run scans in authorized environments only.
- Keep dependencies and toolchain updated.
- Sanitize output logs before sharing or archiving.

Thank you for helping maintain the integrity of Swiss Army Knife.
